🔧 How It Works

Edge Inference

Runs on NVIDIA Jetson Orin modules at each venue gate, processing video feeds at 30fps with YOLOv8 for anonymized person counting.

Privacy First

Anonymised count only — no biometric data, no facial recognition. All processing happens on-device; only aggregate counts leave the edge node.

Refresh Rate

500ms sensor read interval with 10-second aggregation windows pushed to the central dashboard via WebSocket.
